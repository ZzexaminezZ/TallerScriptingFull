Instrucciones:
1. En la pantalla principal aparece un boton en todo el medio, ahí comienza el juego
2. Puedes elegir cualquier de las chicas
3. Al elegir a una de ellas, comenzará el juego 
   1. Taylor(Nice)
   2. Ariana(Medium)
   3. Karol(Hard)
4. La idea es clickear cada una de esas burbujas y así conseguir tanta expectativa como puedas, evitar que baje el nivel de interés de tu cita
5. Para que el juego no tenga un tiempo infinito, se restringe este
6. Ganas cuando consigues una cantidad de interés y expectativa ,y pierdes si tu nivel de interés baja.
* Hay 6 tipos de burbujas que caen
   * Flirt(Amarillo)
   * Love(Rojo)
   * Intelligence(Azul)
   * Intimacy(Verde)
   * Affection(Morado)
   * Boorish(Negro)
Bugs
* Es recomendable ejecutar al momento de abrir Unity ,con una resolución de 1920 x 1080 ya que se puede dañar la ubicación de UI, y como están ubicados los paneles. Además podrían aparecer las burbujas grandes.
* La flecha del termómetro se mueve durante la partida, aunque tiende a quedarse quieta en algunos puntos
* Puede ser que se pase el nivel de Expectativa
